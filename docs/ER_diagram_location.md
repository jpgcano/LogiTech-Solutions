# ER Diagram

The ER diagram for the PostgreSQL schema is included in the repository under:

`data/Diagrama ER de base de datos (pata de gallo).pdf`

If you need the diagram inside the `docs/` folder, copy the file:

```bash
cp "data/Diagrama ER de base de datos (pata de gallo).pdf" docs/ER_diagram.pdf
```
